# AI Resume Analyzer & Job Recommendation System

## Overview

AI Resume Analyzer & Job Recommendation System is a Flask-based web application that helps users analyze resumes, calculate ATS scores, receive AI-powered resume improvement suggestions, generate interview questions, and get suitable job recommendations based on their skills.

The project is designed to assist students, freshers, and job seekers in improving their resumes and increasing their chances of getting shortlisted by recruiters.

## Installation

### Clone Repository

```bash
git clone https://github.com/Poorni2k/AI-Resume-Analyzer-and-job-recommendation-system.git
cd AI-Resume-Analyzer-and-job-recommendation-system
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
